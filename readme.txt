== Description ==
This Plugin will clean your wordpresss blog database from unused data like spam comments, revisions for post and pages

== Installation ==
1. Copy plugin folder (cleanup-wordpress) into wp-content\plugins\
2. Go To your wordpress dashboard and activate plugin Cleanup Wordpress
3. Plugin interface can be found on Tools sections

== How to uninstall Wordpress Cleanup ==
1. Deactivate plugin Cleanup Wordpress
2. Delete folder cleanup-wordpress from wp-content\plugins\
